初识C语言

C语言中%d，等等% 的用法，和意义 - 知乎 (zhihu.com)

常用语句

数据类型

变量、常量

  • static 静态变量，在函数中再次使用时不会重新定义

  • const 修饰的常变量，不能更改但实际上还是一个变量

  • #define 定义的标识符常量

  • 枚举常量

字符串

  • 字符串的结束标志是一个 \0 的转义字符

操作符

  • b = a++ 与 b = ++a，第一个是b先等于a，然后a再加2，第二个是a先加2，然后b等于a
  • 强制类型转换
  • exp? exp1 : exp2 三目操作符 exp? 为真则执行 exp1 ,为假则执行 exp2

关键字

  • extern 声明变量或者函数来自其他文件
  • define 与 typedefine

switch语句

case 是入口，break 是出口，没有 break 从符合条件的 case 处开始往下执行，switch 的条件只能是整数常量

循环语句

for 循环

  • 求阶乘的累加

while 循环

do while 循环

continue 退出本次循环

break 退出循环

函数

定义函数与头文件

  • 通常使用 add.c 与 add.h 来定义函数与头文件，add.c 来存放函数，add.h 来存放函数 函数与变量的声明

定义头文件

使用头文件

函数的定义与声明

  • 函数定义在主函数后可以不用声明
  • 外部函数与变量用 extern 来声明

函数的调用

传值调用

传地址调用

  • 利用指针可以在函数内部对外部变量进行运算赋值
  • 传递数组

函数的递归

  • 函数自己调用自己，可以将复杂问题转变为多个重复的小问题

数组

数组相关性质

  • 字符串数组的长度
  • 一维数组的地址是连续的，每次加上对应数据类型的字节,地址由低到高
  • char 类型的数组，定义方式不同，sizeof() 的结果不同
  • 通过函数传递数组，传递的只是首元素的地址，不能在函数里面通过 sizeof() 来计算元素个数
  • 数组地址与数组首元素地址 数组 arr 是数组首元素地址，sizeof(arr) 除外

二维数组

  • 二维数组的行可以省略，行不可以省略
  • 二维数组的地址是连续的，每次加上对应数据类型的字节

操作符

运算操作符

  • >> 只能作用于整数
    • 算术右移 右边丢弃，左边补原来的符号位
    • 逻辑右移 右边丢弃，左边补0
  • << 只能作用于整数
    • 左移操作 左边丢弃，右边补0,相当于乘以2

  • & 按位与，只能作用于整数

    • 求整数二进制中1的个数

  • | 按位或，只能作用于整数

  • ！ 按位非，只能作用于整数

  • ^ 按位非，只能作用于整数，相异为1，相同为0

    • 利用异或不创建变量来交换变量

逻辑操作符

  • && 逻辑与， a && b && c, 若左边为0， 则右边不再运算
  • || 逻辑或，a && b && c, 若左边为1， 则右边不再运算

符合运算符

  • += a += 1 等价于 a = a + 1
  • -= a -= 1 等价于 a = a - 1

单目操作符

  • 只有一个操作数，称为单目操作符
  • !a 非运算
  • -a 取反
  • &a 取地址操作符
  • *a 解引用操作符，a 为地址找到地址对应的内容
  • &a 取地址操作符
  • sizeof(a) 计算a的大小，单位为字节，sizeof()内部不参与运算
  • ~a 按位取反
  • ++ a++, 先使用后加加， ++a 先加加后使用
  • (int) 强制类型转换

条件操作符

  • exp? exp1 : exp2 , exp? 为1则执行 exp1, 为0则执行 exp2

逗号表达式

  • a = (exp0, exp1, exp2)从左到右执行，整个表达式的的结果是最后一个表达式的结果

表达式求值

隐式表达式

  • 字符和短整型操作数在运算前会进行整型提升，即将其转换为普通整形，再进行运算。整型提升是将其首位扩充到整形的4字节

运算过程

  • a = 01111111, 整型提升: a = 00000000 00000000 00000000 01111111
  • b = 0000011, 整型提升: b = 00000000 00000000 00000000 0000011
  • c = a + b = 00000000 00000000 00000000 1000010 = 10000010
  • 打印时进行运算c = 1000010，整型提升: c = 11111111 11111111 11111111 10000010（补码）
  • c = 11111111 11111111 11111111 10000001（反码）
  • c = 10000000 00000000 00000000 01111110（原码）

运算优先级

指针

  • 指针+/- 指针为元素的个数
  • const int* p = &n 表示 *p 不能被修改， int* const p = &n 表示 p 不能被修改

指针的类型

  • 指针大小都是4/8个字节
  • 指针类型决定了解引用对应内存大小，以及指针+1对应地址

野指针

定义

  • 指针随机不受控制的情况

野指针成因

  • 指针未初始化

  • 指针越界访问

  • 指针指向的空间释放

指针运算

指针 +/- 整数

指针 - 指针

  • 指针 - 指针是元素的个数

指针的关系运算

  • 指针比较大小

    允许指向数组元素的指针与指向数组最后一个元素后面的那个内存位置的指针比较，但是不允许与指向第一个元素之前的那个内存位置的指针进行比较。即只允许大的指针与小的指针比较

指针与数组

  • 数组名 arr 是数组首元素地址,有两个例外
    • &arr 是数组全部数组元素地址
    • sizeof(arr) 是整个数组的大小

二级指针

指针数组

  • int* arr[] = {&a, &b, &b} 存放指针的数组

结构体的定义
