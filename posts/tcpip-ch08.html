IP&域名&DNS

IP 地址: 用于表示不同的机器
DNS(Domain Name System)：是IP地址和域名进行相互转换的系统，其核心是 DNS 服务器。
域名：计算器访问域名，DNS 服务器会将域名解析为 IP 地址，从而实现用户通过易记的域名访问网络资源。

IP 地址与域名直接的相互转换

gethostbyname

☀️

作用:: 将域名转换为 IP 地址

用法：

  • hostname：域名字符串指针

    | 参数 | 作用 |
    | :———— | :———- |
    | hostname | 域名字符串指针 |

注意：无

⭐

示例:: gethostbyname.c

说明:: 将域名转换为 IP 地址

gethostbyaddr

☀️

作用:: 将 IP 地址转换为域名 ^5bc6dd

用法：

  • addr：含有 IP 地址信息的 in_addr 结构体指针。 为了同时传递 IPv4 地址之外的其他信息，该变量的类型声明为 char 的指针
  • len：向第一个参数传递的地址信息的字节数，IPv4 为 4，IPv6 为 16

  • family：传递地址族信息，IPv4 为 AF_INET，IPv6 时为 AF_INET6

    | 参数 | 作用 |
    | :———- | :—- |
    | AF_INET | IPv4 |
    | AF_INET6 | IPv6 |

注意：无
